# KNBSB Hoofdklasse Standen Scraper

Scrapt elk uur automatisch de standen van de Lucky Day Honkbal Hoofdklasse en slaat ze op als `standen.json`.

## JSON ophalen

De standen zijn publiek beschikbaar via:

https://raw.githubusercontent.com/JOUW-GEBRUIKERSNAAM/knbsb-scraper/main/standen.json

Vervang `JOUW-GEBRUIKERSNAAM` door jouw GitHub gebruikersnaam.

## Handmatig starten

Ga naar Actions → KNBSB Standen Scraper → Run workflow om de scraper direct te starten.
